This is a **pre-registration** log: every run is recorded, including the honest
control passes. Verdicts are derived from in-repo ground truth, never hand math.
Ground truth for the holed-plate runs is `sigma_max = Kt_net * sigma_net` with
`Kt_net` from the Peterson/Heywood fit and `sigma_net = P / ((W - d) t)`,
computed by `mechaudit.stress_concentration.plate_with_hole`. The pressure-vessel
controls use `sigma_h = p r / t` and the axial controls `sigma = P / (pi r^2)`.
Tolerance is the repo default (relative 0.5%).

Raw bytes are captured **verbatim** (the `claude -p` stdin warning line that the
CLI emitted on the `sc*` runs is part of the captured artifact and was not
edited out).
