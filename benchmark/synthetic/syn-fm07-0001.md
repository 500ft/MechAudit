## Original Engineering Problem

```text
Find the hoop stress in a thin-walled cylindrical pressure vessel with internal
pressure p = 1.2 MPa, radius r = 50 mm, and wall thickness t = 3 mm.
```

## LLM Prompt

```text
Solve the pressure vessel problem and show your work.
```

## LLM Response

```text
For a thin-walled pressure vessel, the hoop stress is:

sigma_h = p r / t

Substituting p = 1.2 MPa, r = 50 mm, and t = 3 mm:

sigma_h = 25 MPa

The hoop stress is 20 MPa.
```

## Trusted Expected Result

The trusted final answer is:

```text
sigma_h = 1.2 * 50 / 3 = 20 MPa
```

The final answer is correct, but the substitution line in the LLM response says
`25 MPa`. That intermediate value does not follow from the stated formula and
inputs.

## Classified Failure Modes

- `FM-07`: The final answer matches the benchmark expected result, but the
  stated formula and stated inputs recompute to a different value.

## Expected Future Verifier Behavior

The verifier should recompute `sigma_h = p * r / t` using the stated inputs and
flag that the stated substitution value should be `20 MPa`, not `25 MPa`. The
report should make clear that this is a reasoning consistency failure, not a
final-answer failure.
